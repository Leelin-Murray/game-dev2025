Everything we have done with JavaScript on web pages is called frontend coding. Backend coding works in the background and handles data and servers. It is more powerful but harder to learn.

Backends I researched:
• Supabase
• Render
• Firebase
• Netlify

Backends I tried:
• Supabase
• Render

Conclusion:
Supabase and Render are both free and powerful backend platforms. Supabase is good for databases and user login, and Render is good for running backend servers. They take more time to learn, but they are useful for bigger projects. Supabase was easier to understand than Render.
